# Google Search Console Setup

Manual steps for P3-049 and P3-050.

## Verify domain (P3-049)

1. Open [Google Search Console](https://search.google.com/search-console)
2. Add property for `https://anhloom.com`
3. Verify via DNS TXT record or HTML file upload
4. Confirm ownership status is **Verified**

## Submit sitemap (P3-050)

1. In Search Console, open **Sitemaps**
2. Submit `https://anhloom.com/sitemap.xml`
3. Confirm status shows **Success**
4. Re-submit after major route or CMS changes

See also `docs/seo-validation.md` for structured data checks.
